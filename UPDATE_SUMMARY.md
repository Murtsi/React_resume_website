## 🔧 Customization Tips

### Change Colors
Edit `tailwind.config.js` to modify the primary/secondary/accent colors.

### Update Content
- Hero Section: Edit `components/HeroSection.js`
- Experience: Edit `components/ExperienceEducationSection.js`
- Contact: Edit `components/ContactSection.js`

### Add New Sections
1. Create component in `components/NewSection.js`
2. Import in `pages/index.js`
3. Add to the main layout
