# Antti Murtokangas Portfolio

A modern personal resume/portfolio website built with Next.js and Tailwind CSS.

## Live Site

- https://www.anttimurtokangas.com


## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Deployment (Vercel)

- Import the GitHub repo into Vercel
- Ensure build command: `npm run build`
- Output: Next.js default
