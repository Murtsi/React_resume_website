# Antti Murtokangas Portfolio

A modern personal resume/portfolio website built with Next.js and Tailwind CSS. Focused on fiber optic network operations, automation engineering, and professional skills.

## Live Site

- https://www.anttimurtokangas.com

## Highlights

- Clean, recruiter‑friendly layout
- Quick Stats section for instant credibility
- Core Tools strip (iMaster, Altiplano, Optopro Fiber, Keycom, Codesys, Eplan, Cognex, Visual Components)
- Work Experience, Education, and Language Skills
- Contact section with email, LinkedIn, and GitHub
- Dark mode support

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Heroicons

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Deployment (Vercel)

- Import the GitHub repo into Vercel
- Ensure build command: `npm run build`
- Output: Next.js default
